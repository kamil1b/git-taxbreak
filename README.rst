Install
-------

.. code-block:: sh

   pip install git-taxbreak

Usage
-----

.. code-block:: sh

   usage: git-taxbreak [-h] [-u USER] [-U UNIFIED] [-a AFTER] [-b BEFORE]
                    [-o OUTPUT]

   optional arguments:
      -h, --help            show this help message and exit
      -u USER, --user USER
      -U UNIFIED, --unified UNIFIED
      -a AFTER, --after AFTER
      -b BEFORE, --before BEFORE
      -o OUTPUT, --output OUTPUT
